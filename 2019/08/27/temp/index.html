字段名称 类型 说明
banner String 顶部banner位
deviceId Long 设备ID
templateId Long 模板编号
videoUrl String 视频URL
imageUrl String 图片URL
productId Long 商品ID（用于生成二维码，为null时为非商品广告）
deviceNumber String 设备编号HZ_10001
deviceAddress String 设备地址（GPS定位）
deviceAddressInput String 手动录入的设备地址
screenSize String 屏幕尺寸
screenRatio String 屏幕比例
direction int 屏幕方向 1：横屏 2：竖屏
longitude double 经度
latitude double 纬度
secondPrice double 按秒计价 元/秒
dayPrice double 按天计价 元/天
zoneType int 地段类型 0：未录入 1：小区 2：商场
jcvUrl String 实景图
onlineStatus int 在线状态 0：掉线 1：在线
enableStatus int 启用状态 0：未启用 1：启用
onTime date 开机时间
offTime date 关机时间
remark String 备注
input int 是否已完成相关信息录入 0：否 1：是
createTime date 创建时间
updateTime date 修改时间
